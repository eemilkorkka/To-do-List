## Run Locally

Clone the project

```bash
  git clone https://github.com/eemilkorkka/To-do-List.git
```

Go to the project directory

```bash
  cd To-do-List
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
